## Text Elements
***
##### List Item Element
The `<li>` list item element create list items inside:

-   Ordered lists `<ol>`
-   Unordered lists `<ul>`
```html
<ol>  
  <li>Head east on Prince St</li>  
  <li>Turn left on Elizabeth</li>  
</ol>  
  
<ul>  
  <li>Cookies</li>  
  <li>Milk</li>  
</ul>
```
