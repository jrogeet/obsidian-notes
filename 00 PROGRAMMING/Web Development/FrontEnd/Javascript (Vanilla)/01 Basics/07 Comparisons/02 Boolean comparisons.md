# 02 Boolean comparisons

--- 
[[03 Boolean Conversion | [ Boolean Conversion ]]]

```javascript
alert( 2 > 1 ); // true (correct)
alert( 2 == 1 ); // false (wrong)
alert( 2 != 1 ); // true (correct)
```
```javascript
let result = 5 > 4; // assign the result of the comparison
alert( result ); // true
```
